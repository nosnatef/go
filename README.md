# Receipt Processing API (Zhuohong Gu)

### Steps

1. Clone this repository:
   ```sh
   git clone https://github.com/ShiziTech1/go-take-home.git
   cd go-take-home
   ```
2. Install dependencies:
   ```sh
   go mod tidy
   ```
3. Run the server:
   ```sh
   go run api.go
   ```

The server will start on `http://localhost:8080`.
